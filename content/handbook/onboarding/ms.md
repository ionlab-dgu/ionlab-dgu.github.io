# 석사과정 온보딩

목표는 **졸업 논문이 될 연구 하나를 끝까지 끌고 가는 것**입니다.

## 첫 주

- [ ] 프로필 작성 — `content/members/ms-<이름>.md`
- [ ] 랩 서버·GitHub org 계정 세팅
- [ ] 이 핸드북 전체 훑기 (특히 [저자권](../policies/authorship.md), [졸업 요건](../policies/graduation.md))
- [ ] PI와 첫 1:1 — 관심 주제 좁히기
- [ ] 정기 미팅 일정 확정 (주간 진행 미팅 + 격주 1:1)

## 첫 달

- [ ] 관심 분야 논문 10편 읽고 `reading.md`에 정리
- [ ] 베이스라인 1개 재현 — 이것이 첫 마일스톤입니다
- [ ] 연구 프로젝트 폴더 생성 (`pnpm new:project`)
- [ ] 첫 주간 미팅 노트 작성

## 첫 학기

- [ ] 연구 질문을 한 문장으로 쓸 수 있게 되기
- [ ] 첫 실험 사이클 완주 (가설 → 실험 → 해석 → 다음 가설)
- [ ] 국내 학회 또는 워크숍 투고 목표 설정

## 정기 일정

| 주기   | 무엇                                    |
| ------ | --------------------------------------- |
| 주 1회 | 프로젝트 진행 미팅 (노트는 본인이 작성) |
| 격주   | PI와 1:1                                |
| 주 1회 | 리딩 그룹 (돌아가며 발표)               |
