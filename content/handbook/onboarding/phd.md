# 박사과정 온보딩

목표는 **본인의 연구 어젠다를 갖는 것**입니다. 주어진 문제를 푸는 단계에서
문제를 고르는 단계로 넘어가는 과정이라고 생각하면 됩니다.

## 첫 달

- [ ] 프로필 작성 — `content/members/phd-<이름>.md`
- [ ] 핸드북 전체 정독 ([자격시험](../policies/qualification.md), [졸업 요건](../policies/graduation.md) 포함)
- [ ] PI와 연구 방향 합의 — 첫 1~2년의 큰 그림
- [ ] 진행 중인 과제(grant) 중 참여할 것 확인 — 참여율은 프로필에 기록

## 첫 학기

- [ ] 서베이 수준으로 분야 지도 그리기 — `reading.md`가 그 결과물
- [ ] 첫 연구 프로젝트 시작, 상위 학회 투고를 목표로 설정
- [ ] 후배(인턴/석사) 1명 멘토링 시작

## 연차별 이정표

| 시기 | 이정표 |
| --- | --- |
| 1년차 | 분야 파악, 첫 투고 |
| 2년차 | 자격시험 통과, 첫 주저자 논문 |
| 3년차 | 연구 어젠다 확립, 2편째 |
| 4년차~ | 학위논문 구성, 마무리 |
